# PicPick

## Overview

PicPick is a screen capture and image editing tool designed for Windows users. It provides a range of functionalities for capturing, annotating, and editing screenshots, supporting various workflows related to visual content creation and documentation.

## Key Features

**Screen capture with multiple modes including full screen, active window, and region selection**  
**Built-in image editor with drawing tools, color picker, and effects**  
**Support for exporting images in common formats such as PNG, JPEG, and BMP**  
**Color palette and ruler tools for precise design work**  
**Integration of screen magnifier to assist with detailed editing**  
**Options for sharing captured images via clipboard or file saving**  
**Lightweight and responsive interface tailored for efficiency**  

## Why PicPick?

PicPick is chosen for its straightforward interface and consistent performance on Windows platforms.
